# Student Management System

## 📖 Project Overview
The Student Management System is a web application developed to simplify the management of student information. It allows users to add, update, view, search, and delete student records through an easy-to-use interface.

## 🚀 Features
- User Login Authentication
- Student Dashboard
- Add Student Details
- View Student Records
- Update Student Information
- Delete Student Records
- Search Students
- Responsive User Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- VS Code
- Git & GitHub


## ⚙️ Installation
1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Open `index.html` in your browser.

## ▶️ Usage
1. Open the application.
2. Log in to the system.
3. Manage student records using the dashboard.
4. Add, edit, search, or delete student details.
